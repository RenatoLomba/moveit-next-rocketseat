<h2 id="idex">📖 About it</h2>

<br>

<p>
    &nbsp;&nbsp;&nbsp;&nbsp;<strong>Move.it</strong> é um aplicativo, rodando no navegador, com acesso livre, baseado na técnica de Pomodoro, para ajudar na saúde física de

</p>
